ZXEI Icons

一个基于SVG的Vue图标组件库，提供丰富的图标集合

GitHub NPM

安装

npm install zxei-icons

使用方法

全局注册

import { createApp } from 'vue';
import App from './App.vue';
import ZxeiIcons from 'zxei-icons';

const app = createApp(App);
app.use(ZxeiIcons);
app.mount('#app');

按需引入

import { ArrowDownFill } from 'zxei-icons';

图标分类

箭头类
ArrowDownFill
